# Curiositas ClawKit Demo

Frontend demo per Curiositas ClawKit v1 - Costruito con Next.js, Tailwind CSS e Anime.js.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Anime.js v4 per animazioni

## Installazione

```bash
npm install
npm run dev
```

## Deploy su Vercel

```bash
npx vercel login
npx vercel --prod
```
